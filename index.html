🥧 Black Whisk Setup

One-time setup to connect your shared database. Free, no credit card. Takes 2 minutes. Do this once, then share the file with your baker — they'll use the same ID + key.

1
Go to jsonbin.io → click Sign Up (free account)
2
Click Bins in the sidebar → + Create Bin → paste {} → click Create
3
Copy the Bin ID shown at the top (looks like 64abc123…)
4
Click your profile icon → API Keys → copy your Secret Key
5
Paste both below and hit Save & Open App

Your credentials stay in this browser only. Share this HTML file + the Bin ID + Secret Key with your baker so they connect to the same data.
